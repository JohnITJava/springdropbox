DropBox for storage files.
Functions:
 - Registration
 - Authentication
 - File upload
 - File download

Technologies:
 - Spring Boot
 - Spring Data
 - Spring Security
 - MySql
 - Flyway
 - Lomboc
 - DevTools

 How to start?
  - Create your DB scheme without tables.
  - Resources -> private.properties - write your configs for db. 
  In maven for flyway also write your configs. 
  - Flyway -> migrate
  - Run project
